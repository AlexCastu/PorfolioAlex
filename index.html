Quiz

Descripción

Proyecto para TheBridge para finalizar la parte del Front. Es un juego de preguntas. Las preguntas las consultamos mediante el consumo de una API de preguntas, posteriormente insertamos preguntas desde Firebase. La misma plataforma la usamos para dar de alta el usuario y la autentificación de cada usuario, guardando puntuación en la base de datos.

Cronometro

Descripción

Aplicación de cronómetro, usamos el LocalStorage para guardar las sesiones en local y así poder consultarlas a posteriori. Se guardan las vueltas siempre que guardas, tienes un temporizado de 10 segundos y puedes pausar y restablecer las veces que quieras.

Ramp UP

Descripción

Primer ejercicio grande en TheBridge, consta de una página en la que desarrollamos los conocimientos adquiridos en la primera fase del BootCamp. Solo se podría usar HTML y CSS.

API Dog

Descripción

Primeras prácticas en JS de consumo de APIS usando "Fetch". Se trata de una aplicación que consume una API de perritos, en la cual hacemos varias consultas dependiendo de lo que busque el usuario. Consta de una parte que muestra todas las razas de perro, otra de una imagen aleatoria de la API y un select para el usuario que muestre la foto de la búsqueda.
